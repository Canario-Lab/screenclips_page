개인정보 처리방침

시행일: 2026년 6월 16일

버전: 1.0

1) 데이터 관리자

개인정보 관리자는 Canario-Lab / ScreenClips 앱입니다.

ScreenClips는 스크린샷을 Clip과 라이브러리로 정리하는 로컬 iOS 앱입니다. 앱은 사용자 계정이나 회원가입을 요구하지 않습니다.

2) 로컬에 저장되는 데이터

앱에서 만든 데이터에는 Clip, 라이브러리, 메모, 아이콘, 스크린, 언어 설정, Face ID 설정, 백업 정보가 포함됩니다. 이 데이터는 사용자의 기기 또는 메인 앱과 공유 확장 기능이 함께 쓰는 로컬 App Group 컨테이너에 저장됩니다.

이 데이터는 자동으로 Canario-Lab 또는 Canario-Lab 서버로 전송되지 않습니다.

3) 사진, 스크린, 공유 확장 기능

앱은 사용자가 선택한 이미지를 Clip에 추가할 때만 사진 접근 권한을 사용합니다. ScreenClips는 앱 안에서 사용할 자체 최적화 복사본을 저장합니다.

Clip에서 스크린을 제거해도 ScreenClips 데이터에서만 제거됩니다. 사용자의 사진 보관함에 있는 원본 사진은 삭제되지 않습니다.

공유 확장 기능을 통해 사용자는 시스템 공유 시트에서 선택한 이미지를 ScreenClips로 보낼 수 있습니다. 확장 기능은 App Group group.com.canario-lab.screenclip.shared를 통해 같은 로컬 저장소에 데이터를 기록합니다.

4) 백업, 내보내기, 공유

백업은 Clip, 라이브러리, 스크린, 앱 설정이 담긴 로컬 .screenclipbackup 파일을 만듭니다. 이 파일은 사용자가 선택한 시스템 방식으로 저장하거나 공유합니다.

백업을 복원하면 앱의 현재 로컬 상태가 선택한 파일의 데이터로 대체됩니다. 앱 밖으로 공유한 백업 파일, PDF 및 기타 자료를 보관하고 보호할 책임은 사용자에게 있습니다.

5) Face ID 및 기기 보안

사용자가 Face ID로 앱 잠금을 켜면 인증은 iOS의 LocalAuthentication을 통해 처리됩니다. ScreenClips는 생체 인식 데이터를 받거나 저장하지 않습니다.

앱 데이터는 로컬에 저장되므로 암호, Face ID 또는 기타 시스템 보안 기능으로 기기 자체를 보호하는 것이 중요합니다.

6) Premium 구매, Apple, RevenueCat

ScreenClips는 App Store에서 한 번 구매하는 Lifetime 방식으로 유료 Premium 접근을 제공할 수 있습니다. 앱은 월간 또는 연간 구독과 무료 체험을 제공하지 않습니다.

결제는 Apple이 App Store와 StoreKit을 통해 처리합니다. ScreenClips는 상품 표시, ScreenClips Pro 권한 확인, 구매 복원, 앱 내 기술적 RevenueCat ID 표시를 위해 RevenueCat을 사용합니다.

이 과정에서 Apple과 RevenueCat은 거래 식별자, 제품 식별자, Premium 접근 상태, 접근 권한 확인에 쓰이는 익명 앱 사용자 식별자 같은 기술적 구매 데이터를 처리할 수 있습니다.

Canario-Lab은 사용자의 결제 카드 정보를 처리하지 않습니다.

7) 데이터 수신자

데이터는 사용자가 선택한 기능 또는 구매 처리에 필요한 범위에서만 외부 서비스에 의해 처리될 수 있습니다.

  • App Store, StoreKit, 결제, 시스템 파일 선택기, 사진, Face ID, 공유를 위한 Apple.
  • 상품 처리, Premium 확인, Lifetime 구매 복원을 위한 RevenueCat.
  • 사용자가 PDF나 백업을 내보내거나 ScreenClips 밖으로 파일을 공유할 때 선택한 시스템 제공자.
  • 관련 법률에 따라 요구되는 경우 공공 기관.

8) 보관 기간

ScreenClips에 저장된 데이터는 사용자가 삭제하거나, 다른 백업을 복원하거나, 앱을 제거하거나, iOS 시스템 방식으로 데이터를 제거할 때까지 로컬에 남습니다.

앱 밖으로 내보낸 백업, PDF, 파일은 사용자와 해당 파일이 전송된 서비스의 관리 아래에 있습니다.

Apple과 RevenueCat이 처리하는 구매 데이터는 해당 서비스의 정책에 따라 보관됩니다.

9) 사용자 권리

Canario-Lab이 지원 문의 등 특정 상황에서 사용자의 개인정보를 처리하는 경우, 사용자는 접근, 수정, 삭제, 처리 제한, 처리 반대를 요청할 수 있으며 관할 감독 기관에 이의를 제기할 수 있습니다.

개인정보 관련 문의는 App Store 또는 앱에 표시된 연락 채널을 이용해 주세요.

10) 방침 변경

이 개인정보 처리방침은 ScreenClips 기능, 결제 모델 또는 법적 요구 사항이 바뀌면 업데이트될 수 있습니다. 최신 버전은 앱에서 확인할 수 있습니다.
